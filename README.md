# the_coordinate
The coordinate connects everything

## FRONTEND

### Tech Stack

- Framework: Next.js
- Library: React
- Language: TypeScript
- Styling: Tailwind CSS


### Local Deployment

Set up commands

```
git clone https://github.com/jean-johnson-zwix/the_coordinate.git\
cd the_coordinate\frontend
npm install
```

Run commands

```
npm run dev
```
